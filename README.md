# Cách chạy project

1. Cài Python
2. Cài MySQL
3. Tạo database tên Learn_English_Website mật khẩu 123456 (tự nghiên cứu nha anh em)
4. pip install -r requirements.txt (gõ vào terminal, chọn qua màn hình command prompt rồi gõ lệnh này để tải thư viện cần thiết về)
5. python app.py (sau khi download xong thì gõ lệnh này)
